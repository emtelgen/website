Web Creation

Creation of this website using HTML, CSS, and JavaScript. This website is used as a tool to display all of my other projects I have worked on as well.

Creation Procedure

Step One

Creating a navigation bar that remains throughout each page. This navigation bar features faded text upon hovering over the items themselves and a button element to access a PDF of my Resume.

Step Two

Using the flexbox tool to format the individual pages appropriately for viewing. I set up elements in rows and columns throughout the pages to prevent overlap and make viewing easiest.

Step Three

Adding visually appealing elements to make the website look proper. This included hovering text over images, matching a color theme, adding elements to divide and shape pages, as well as lots more.

Step Four

Hosting the website on GitHub so that users can access and view the website.

Additional Modifications

Additional modifications that could be used to make this website better would include more interactivity between the site and the user. Elements such as zooming pictures or page headers that remained at the top of the page until a new section is scrolled to.
