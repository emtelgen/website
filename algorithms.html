Sorting and Searching

Used Python and Scala to create github respositories with different sorting and searching algorithms including different tree traversal algorithms such as postorder and inorder traversals, binary search algorithm, and sorting algorithms such as selection sort and insertion sort. I have displayed a few of these programs below.

Creation Procedure

Binary Search

A searching algorithm coded iteratively that determines the position of an element in a sorted array. Notable attributes of the algorithm include determining if the target item is less than or greater than the current indexed element, and repeating that process until the target is found or determined to not exist. Coded in Python.

Pre-Order Traversal

A type of tree traversal that recursively searches through a tree. The algorithm adds the root to the list, followed by the left node and subsequently the right node. Coded in Scala.

Merge Sort

A sorting algorithm that recursively divides the array in half until there can be no more splits made. These subarrays are merged back into a full array in the correct order. Coded in Python.

Post-Order N-ary Tree Traversal

A traversal algorithm that searches through a N-ary tree starting at the left node, then the right node, then adding the root lastly. Coded in Python.
